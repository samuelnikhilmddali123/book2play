# Book2Play - Sports Ground Booking Platform

A modern, premium sports ground booking web application built with Next.js, Tailwind CSS, and Framer Motion.

## 🚀 Deployment to Vercel

1.  **Push to GitHub**: (Already completed)
2.  **Import to Vercel**:
    *   Go to [vercel.com](https://vercel.com).
    *   Click **"Add New"** > **"Project"**.
    *   Import the `book2play` repository.
3.  **Configure Project**:
    *   **Root Directory**: Set this to `frontend`.
    *   **Framework Preset**: Next.js.
4.  **Deploy**: Click the **Deploy** button.

## 🛠️ Tech Stack
- Frontend: Next.js 15 (App Router)
- Styling: Tailwind CSS v4
- Animations: Framer Motion
- Icons: Lucide React

## 📂 Features
- **Dynamic Home Page**: High-fidelity sports categories (Cricket, Football, Badminton, Volleyball).
- **Interactive Listings**: Filterable ground search.
- **Booking Flow**: Secure checkout and slot management.
- **Admin Dashboard**: Revenue and ground management analytics.
